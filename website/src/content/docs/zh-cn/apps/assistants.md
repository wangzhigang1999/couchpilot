<span class="process-chip">Claude.exe · CherryStudio.exe</span>

保留通用鼠标操作，同时加入与 Codex 相同的白名单语音编辑流程。

| 按键 | 动作 | 键盘快捷键 |
| --- | --- | --- |
| <kbd>B</kbd> | 关闭浮层 / 返回 | Escape |
| <kbd>L3</kbd> | 查找 | Ctrl + F |
| 按过 <kbd>Y</kbd> 后，轻按 <kbd>A</kbd> | 发送语音输入的消息 | Enter |
| 按过 <kbd>Y</kbd> 后，轻按或按住 <kbd>B</kbd> | 删除一个或连续删除 | Backspace |
| <kbd>RT</kbd> + <kbd>A</kbd> | 随时发送 | Enter |

离开临时语音编辑状态后，A 仍然是鼠标左键，B 仍然是 Escape。
