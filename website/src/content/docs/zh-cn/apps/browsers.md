<span class="process-chip">chrome.exe · msedge.exe · firefox.exe</span>

肩键切换标签，摇杆按下直达地址栏和新标签。

| 按键 | 动作 | 键盘快捷键 |
| --- | --- | --- |
| <kbd>B</kbd> | 后退 | Alt + Left |
| <kbd>LB</kbd> | 上一个标签 | Ctrl + Shift + Tab |
| <kbd>RB</kbd> | 下一个标签 | Ctrl + Tab |
| <kbd>L3</kbd> | 聚焦地址栏 | Ctrl + L |
| <kbd>R3</kbd> | 新建标签 | Ctrl + T |
