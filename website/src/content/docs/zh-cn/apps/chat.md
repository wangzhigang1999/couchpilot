<span class="process-chip">QQ.exe · Weixin.exe · WeChat.exe</span>

通过手柄查找和关闭浮层，并且只在这个白名单 profile 中启用已经验证的语音编辑流程。

| 按键 | 动作 | 键盘快捷键 |
| --- | --- | --- |
| <kbd>B</kbd> | 关闭浮层 / 返回 | Escape |
| <kbd>L3</kbd> | 查找 | Ctrl + F |
| 按过 <kbd>Y</kbd> 后，轻按 <kbd>A</kbd> | 发送语音输入的消息 | Enter |
| 按过 <kbd>Y</kbd> 后，轻按或按住 <kbd>B</kbd> | 删除一个或连续删除 | Backspace |
| <kbd>RT</kbd> + <kbd>A</kbd> | 随时发送 | Enter |

:::caution[Enter 发送]
这个白名单假设应用使用 Enter 发送。离开临时语音编辑状态后，A 仍然是鼠标左键，B 仍然是 Escape。
:::
