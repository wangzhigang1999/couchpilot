| 按键 | 动作 | 键盘快捷键 |
| --- | --- | --- |
| <kbd>LB</kbd> | 上一个标签 | Ctrl + Shift + Tab |
| <kbd>RB</kbd> | 下一个标签 | Ctrl + Tab |
| <kbd>L3</kbd> | 查找 | Ctrl + F |
| <kbd>R3</kbd> | 新建文档 | Ctrl + N |

:::caution[不抢焦点]
A 仍是鼠标左键，CouchPilot 不会自动聚焦编辑区。
:::
