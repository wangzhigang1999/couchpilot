| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>LB</kbd> | Previous tab | Ctrl + Shift + Tab |
| <kbd>RB</kbd> | Next tab | Ctrl + Tab |
| <kbd>L3</kbd> | Find | Ctrl + F |
| <kbd>R3</kbd> | New document | Ctrl + N |

:::caution[No focus stealing]
A remains the left mouse button. CouchPilot never focuses the editor automatically.
:::
