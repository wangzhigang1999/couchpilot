<span class="process-chip">Claude.exe · CherryStudio.exe</span>

Keep the global pointer controls while adding the whitelisted voice-edit flow used by Codex.

| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>B</kbd> | Close overlay / back | Escape |
| <kbd>L3</kbd> | Find | Ctrl + F |
| After <kbd>Y</kbd>, tap <kbd>A</kbd> | Send the dictated message | Enter |
| After <kbd>Y</kbd>, tap or hold <kbd>B</kbd> | Delete one or keep deleting | Backspace |
| <kbd>RT</kbd> + <kbd>A</kbd> | Send at any time | Enter |

Outside the temporary voice-edit state, A remains the left mouse button and B remains Escape.
