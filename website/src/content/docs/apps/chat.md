<span class="process-chip">QQ.exe · Weixin.exe · WeChat.exe</span>

Find and close overlays from the gamepad, with the tested voice-edit flow enabled only for this whitelisted profile.

| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>B</kbd> | Close overlay / back | Escape |
| <kbd>L3</kbd> | Find | Ctrl + F |
| After <kbd>Y</kbd>, tap <kbd>A</kbd> | Send the dictated message | Enter |
| After <kbd>Y</kbd>, tap or hold <kbd>B</kbd> | Delete one or keep deleting | Backspace |
| <kbd>RT</kbd> + <kbd>A</kbd> | Send at any time | Enter |

:::caution[Enter-to-send]
This whitelist assumes the app uses Enter to send. Outside the temporary voice-edit state, A remains the left mouse button and B remains Escape.
:::
