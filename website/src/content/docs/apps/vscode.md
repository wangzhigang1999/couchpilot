| Control | Action | Keyboard shortcut |
| --- | --- | --- |

| <kbd>LB</kbd> | Previous editor | Ctrl + Shift + Tab |
| <kbd>RB</kbd> | Next editor | Ctrl + Tab |
| <kbd>L3</kbd> | Command palette | Ctrl + Shift + P |
| <kbd>R3</kbd> | Quick open | Ctrl + P |
