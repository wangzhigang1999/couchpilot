<span class="process-chip">Typeless.exe</span>

Y starts voice input, B closes the overlay, and all other controls keep their global mouse behavior.

| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>Y</kbd> | Start voice input | Right Alt |
| <kbd>B</kbd> | Close overlay | Escape |

:::note
This mapping uses the physical **right Alt** key, not left Alt.
:::
