This profile adds only dependable find and dismiss actions so it does not collide with the IDE's extensive shortcuts.

| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>B</kbd> | Close menu / back | Escape |
| <kbd>L3</kbd> | Find in current file | Ctrl + F |
