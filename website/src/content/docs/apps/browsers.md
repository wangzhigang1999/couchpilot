<span class="process-chip">chrome.exe · msedge.exe · firefox.exe</span>

Use the shoulder buttons for tabs and press the sticks for the address bar and a new tab.

| Control | Action | Keyboard shortcut |
| --- | --- | --- |
| <kbd>B</kbd> | Back | Alt + Left |
| <kbd>LB</kbd> | Previous tab | Ctrl + Shift + Tab |
| <kbd>RB</kbd> | Next tab | Ctrl + Tab |
| <kbd>L3</kbd> | Focus address bar | Ctrl + L |
| <kbd>R3</kbd> | New tab | Ctrl + T |
