## Design principles

<CardGrid>
  <Card title="Direct" icon="rocket">Move the pointer immediately. There is no mode to enter first.</Card>
  <Card title="Predictable" icon="seti">An app profile overrides only the controls documented on its page.</Card>
  <Card title="Safe" icon="shield-check">Chat apps never auto-send, and app switching never steals input focus.</Card>
</CardGrid>
